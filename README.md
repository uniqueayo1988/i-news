# i-news

## Project setup
```
git clone https://github.com/uniqueayo1988/i-news.git
```
```
cd i-news
```
```
npm install
```

### To serve app for development
```
npm run serve
```

### To view app from server
```
http://localhost:8080/
```
Click `Signup` to register and be redirected to the blog view page
